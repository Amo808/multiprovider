# Тестовый документ для RAG

Это тестовый документ для проверки работы RAG (Retrieval Augmented Generation).

## Раздел 1: Введение

RAG - это метод дополнения языковых моделей внешними знаниями. Он позволяет моделям отвечать на вопросы, используя информацию из загруженных документов.

## Раздел 2: Как работает RAG

1. Документ разбивается на части (чанки)
2. Каждый чанк преобразуется в вектор (эмбеддинг)
3. При запросе ищутся похожие чанки
4. Найденный контекст передается в LLM

## Раздел 3: Преимущества

- Актуальная информация
- Снижение галлюцинаций
- Прозрачность источников
